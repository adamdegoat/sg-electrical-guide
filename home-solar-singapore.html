Home solar in Singapore — the basics

⚡ Evergrow Electrical Service General guide

Singapore gets plenty of sun, and rooftop solar can genuinely cut a household's bill. But who can actually install it depends a lot on where you live — so let's clear that up first.

Can you even install it?

  • Landed homes — yes, this is where most private rooftop solar happens. You own the roof, so it's your call.
  • Condos & apartments — the roof is shared, so it's an MCST decision for the whole building, not an individual unit.
  • HDB — solar is handled at the block/town level (the national SolarNova programme), not by individual flats.

So for a typical homeowner asking about their own panels, this really applies to landed property.

How it connects

Panels on the roof feed an inverter, which ties into your home's electrical system and the grid. Any excess you generate can be exported. Because it connects to the grid, the installation must be done by an EMA-licensed professional and registered with the utility (SP Group) — it's not a plug-and-play job.

Rough numbers

Cost scales with system size (measured in kWp). A small landed-home system is a five-figure investment; bigger systems cost more but generate more. Payback typically runs several years, after which the savings are yours. Get a proper site assessment — roof orientation, shading and your actual usage change the maths a lot.

Worth knowing

A solar install is as much an electrical job as a panel job — the inverter wiring, isolation, earthing and grid connection all have to be done correctly and certified. Make sure whoever installs it is properly licensed for the electrical side, not just selling panels.
